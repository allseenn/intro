Задача 2. Составьте блок-схему на основе псевдокода

```pseudo
array = [2, 5, 13, 7, 6, 4]
index = 0
sum = 0
mean = 0
while (index < array.length) do
    sum += array[index]
    index += 1
mean = sum / array.length
print(mean)
```
